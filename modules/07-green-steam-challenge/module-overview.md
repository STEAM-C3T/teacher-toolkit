# Module 7: Green STEAM Challenge

## Module Overview

### Learning outcomes (DigComp 2.2)
- Scope a small app addressing a sustainability/SDG problem.
- Integrate HTML, CSS, and JS to produce a functional prototype.
- Collect user feedback and plan iterations.

### Sequence
1. Identify a focused problem and users; define one key interaction.
2. Prototype build: minimal UI, state, and output.
3. Peer testing; gather feedback; plan improvements.
4. Iterate and present.
