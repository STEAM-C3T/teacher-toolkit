# Module 1: Introduction to the Web

## Assessment Rubric — Introduction to the Web

Criteria (DigComp 2.2 mapping):

1. Understanding the web model (Information & data literacy)

- Emerging: Can state that a browser shows pages.
- Proficient: Explains browser–server–response with an example.
- Exemplary: Illustrates flow and implications (latency, caching).

2. Semantic structure (Digital content creation)

- Emerging: Uses tags inconsistently.
- Proficient: Correct header/main/footer; headings in order.
- Exemplary: Adds nav/sections appropriately; explains choices.

3. Accessibility basics (Safety & inclusion)

- Emerging: Omits language or alt text; unclear links.
- Proficient: Has `lang` attribute; meaningful links.
- Exemplary: Checks keyboard navigation; adjusts for clarity.

Teacher reflection:

- What evidence shows outcomes achieved?
- What adaptations supported varied learners?
- What to improve next time?
