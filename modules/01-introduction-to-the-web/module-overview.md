## Learning outcomes (aligned to DigComp 2.2)
- Describes how the web works (browser–server–response).
- Identifies parts of an HTML document and the role of semantic tags.
- Creates a minimal, accessible HTML page.

## Sequence
1. Activate prior knowledge: “What happens when you type a URL?”
2. Demo: View Source and semantic structure.
3. Guided practice: build minimal page.
4. Independent task: first webpage with header/main/footer.
5. Share & reflect.
