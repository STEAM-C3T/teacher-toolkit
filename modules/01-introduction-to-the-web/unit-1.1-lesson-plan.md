# Module 1: Introduction to the Web

## Unit 1.1 – What is the Web & How It Works

## Outcomes

- Explain request–response flow.
- Identify `<html>`, `<head>`, `<body>` in a simple page.

## Lesson flow

- Intro (5’): prior knowledge & goal.
- Demo (10’): navigate to a page, View Source.
- Guided (15’): highlight semantic landmarks.
- Practice (20’): fill-in minimal page template.
- Share (5’): pairs present & reflect.

## Materials

- DPK Unit: https://github.com/STEAM-C3T/digital-proficiency-kit/blob/main/modules/01-introduction-to-the-web/units/unit-1.1-what-is-the-web.md
- Example: https://github.com/STEAM-C3T/digital-proficiency-kit/blob/main/modules/01-introduction-to-the-web/examples/minimal-page.html

## Differentiation

- Provide a labelled diagram for visual learners; oral explanation variant.
- Extension: simple diagram of the flow (browser → server → response).

## Accessibility

- Emphasise headings and landmarks; avoid presentational `<div>` for structure.
- Validate language attribute and document outline.

## Assessment

- Exit ticket: label the three core HTML elements and their purpose.
- Criteria: correctness of structure; clarity of explanation.
