# Module 3: CSS Styling & Layout

Metadata

- Module: 03 • Version: 2025-11-18 • Audience: Lower/upper secondary

## Module Overview

Teachers help students establish a visual system (typography, colour, spacing) and implement responsive layouts using Flexbox/Grid. Accessibility (contrast, focus, readable scale) is integral.

## Learning Outcomes (DigComp 2.2)

- Applies CSS to create clear hierarchy (type scale), spacing, and colour system.
- Constructs responsive layouts via Grid/Flexbox and mobile-first media queries.
- Evaluates contrast and focus visibility; maintains readability across breakpoints.

### Alignment (DigComp areas)

- Digital content creation (3.1, 3.2)
- Safety & inclusion (4.1 accessibility basics)

## Sequence & Timing (60–120 min)

1. Styling basics (15–25): selectors, variables, type scale, spacing, colour.
2. Layout (20–35): Grid vs Flexbox; breakpoints; card grid demo.
3. Guided practice (15–25): style a simple page with consistent hierarchy.
4. Independent task (10–25): responsive grid and/or navigation.
5. Share & reflect (5–10): contrast/focus checks; before/after screenshots.

## Materials & Setup

- Browser + editor; colour contrast tool (built-in DevTools or online checker).

## Accessibility & Inclusion

- Aim for WCAG AA contrast for body text; never remove focus outlines.
- Allow zoom and avoid fixed pixel sizes where possible.

## Differentiation

- Scaffold: Provide a style guide snippet (CSS custom properties) and base layout.
- Extension: Add responsive nav, print stylesheet, or prefers-reduced-motion support.

## Assessment & Evidence

- Evidence: Styled HTML/CSS + screenshots at two widths + short rationale.
- Use rubric: See [assessment-rubric-03.md](./assessment-rubric-03.md).
