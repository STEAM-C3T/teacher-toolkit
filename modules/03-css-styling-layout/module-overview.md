# Module 3: CSS Styling & Layout

## Module Overview

## Learning outcomes (DigComp 2.2)

- Apply CSS selectors to style content with accessible contrast and typography.
- Construct responsive layouts using Grid/Flexbox and media queries.

## Sequence

1. Styling basics: selectors, colour, type, spacing.
2. Layout: Grid/Flexbox; mobile‑first media queries.
3. Guided practice: style a simple page; build responsive card grid.
4. Independent task; share and reflect.
