# Module 3: CSS Styling & Layout

## Unit 3.1 – Styling Basics — Selectors, Colour, Typography, Box Model

## Outcomes

- Use element/class selectors effectively.
- Set base type scale and spacing; explain box model impacts.

## Lesson flow

- Warm‑up (5’): examine an unstyled HTML page.
- Demo (10’): selectors; colour/contrast; type scale.
- Guided (15’): apply styles to establish hierarchy and spacing.
- Practice (20’): style “About”/portfolio page.
- Share (5’): focus/hover accessibility check.

## Differentiation

- Provide a palette and starter CSS for novices; extension: custom variable theme.

## Accessibility

- Ensure :focus-visible styles; maintain 4.5:1 contrast (body text).

## Assessment

- Checklist: hierarchy clear, spacing consistent, focus states present.
