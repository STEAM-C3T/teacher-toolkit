## Learning outcomes (DigComp 2.2)
- Represent small datasets visually with Canvas or SVG.
- Explain scaling from data values to pixel sizes.

## Sequence
1. Intro to data arrays and simple scaling.
2. Drawing bars/points; adding labels and titles.
3. Guided practice: modify dataset and labels.
4. Independent task; reflect on chart choice.
