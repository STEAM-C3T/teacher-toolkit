# Module 5: Data & Visualization

Metadata

- Module: 05 • Version: 2025-11-18 • Audience: Lower/upper secondary

## Module Overview

Students represent small datasets visually using Canvas or SVG, focusing on mapping values to positions/sizes and adding clear labels. Visuals are paired with an HTML table for accessibility and verification.

## Learning Outcomes (DigComp 2.2)

- Maps numeric data to visual encodings (length, position, colour) with labels.
- Explains scaling choices and axis labelling for simple charts.
- Pairs visualizations with tables to support accessibility and accuracy.

### Alignment (DigComp areas)

- Digital content creation (3.1, 3.2)
- Information/data literacy (1.2)

## Sequence & Timing (60–90 min)

1. Scaling (15): map data range → pixel range; choose chart type.
2. Draw & label (20–30): bars/points, axes, titles, units.
3. Guided practice (10–20): modify dataset and labels; test extremes.
4. Independent (10–20): build own dataset viz + companion table.
5. Reflect (5): explain encoding choice and limitations.

## Materials & Setup

- Browser + editor; optional spreadsheet for quick dataset prep.

## Accessibility & Inclusion

- Provide a companion HTML table with headers and a caption.
- Avoid relying on colour alone; add labels or patterns.

## Differentiation

- Scaffold: Provide starter Canvas/SVG boilerplate with helpers.
- Extension: Add hover tooltips or dataset switcher; consider simple legend.

## Assessment & Evidence

- Evidence: Chart screenshot + HTML/JS + companion table + short rationale.
- Use rubric: See [assessment-rubric-05.md](./assessment-rubric-05.md).
