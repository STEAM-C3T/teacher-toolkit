# Module 4: JavaScript Essentials

## Assessment Rubric — JavaScript Essentials

1) Event handling (Problem solving)
- Emerging: Inline handlers or fragile logic.
- Proficient: addEventListener; clear logic paths.
- Exemplary: Modular functions; edge cases considered.

2) State & rendering (Digital content creation)
- Emerging: DOM mutated ad‑hoc; inconsistent UI state.
- Proficient: State variable + render(); predictable updates.
- Exemplary: Clean separation; explainable state shape.

3) Accessibility (Safety & inclusion)
- Emerging: Missing labels and live regions.
- Proficient: Labeled inputs; aria‑live for dynamic text.
- Exemplary: Thoughtful focus management; keyboard friendly.
