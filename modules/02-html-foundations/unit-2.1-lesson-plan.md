# Module 2: HTML Foundations

## Unit 2.1 – Building Content — Headings, Lists, Links, Images

## Outcomes

- Use headings (H1–H3), paragraphs, lists, links, images with alt text.

## Lesson flow

- Activate (5’): good vs poor content examples.
- Demo (10’): semantics and alt text qualities.
- Guided (15’): add list + link + image with descriptive alt.
- Practice (20’): create a small topic page.
- Share (5’): peer check: alt text & link clarity.

## Materials

- DPK Unit: [Unit 2.1 — Building Content](https://github.com/STEAM-C3T/digital-proficiency-kit/blob/main/modules/02-html-foundations/units/unit-2.1-building-content.md)

## Differentiation

- Provide alt‑text prompts; challenge: use figure/figcaption.

## Accessibility

- Avoid “click here”; ensure unique H1; descriptive alt.

## Assessment

- Checklist: headings order, alt text present/meaningful, link clarity.
