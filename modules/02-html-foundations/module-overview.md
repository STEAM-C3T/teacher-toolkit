## Learning outcomes (DigComp 2.2)
- Construct semantic content with headings, lists, links, images.
- Create forms with labels; present small datasets in tables.

## Sequence
1. Content building blocks; semantics and alt text.
2. Forms and labels; table structure.
3. Guided practice: topic page + student survey.
4. Independent task; share and reflect.

## Teacher notes
- Accessibility: heading order, alt text, label–input association.
- Differentiation: provide skeleton HTML; extensions with anchors/TOC.
- Assessment: criteria for semantic correctness and clarity.
